# Amrit Sagar Website Deployment Guide

## Deployment Options

### 1. Netlify (Recommended)
- **Free tier available**
- **Automatic HTTPS**
- **Continuous deployment**
- **Custom domain support**

### 2. Vercel
- **Free tier available**
- **Zero-config deployment**
- **Automatic HTTPS**
- **Global CDN**

### 3. GitHub Pages
- **Free hosting**
- **GitHub integration**
- **Automatic HTTPS**
- **Simple setup**

### 4. Traditional Hosting
- **Any web server**
- **FTP/SFTP upload**
- **Manual deployment**
- **Full control**

## Quick Deploy Options

### Option 1: Netlify Drag & Drop
1. Build the website files
2. Drag the entire folder to Netlify
3. Get live URL instantly

### Option 2: Vercel CLI
```bash
npm i -g vercel
vercel --prod
```

### Option 3: GitHub Pages
1. Push to GitHub
2. Enable Pages in repository settings
3. Select source branch
